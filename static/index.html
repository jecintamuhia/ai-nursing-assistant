🏗 System Architecture — How It Works
1
User uploads PDF(s)
Nursing textbooks, lecture notes, clinical guidelines dropped into the web UI
FastAPI · python-multipart
↓
2
Document loading & chunking
PDFs are parsed page-by-page and split into 800-character overlapping chunks to preserve context
PyPDFLoader · RecursiveCharacterTextSplitter
↓
3
Semantic embeddings
Each chunk is converted into a 1536-dimensional vector that captures its clinical meaning
OpenAI text-embedding-ada-002
↓
4
Vector database (FAISS)
Vectors stored locally in a FAISS index. At query time, the top-5 most semantically similar chunks are retrieved
FAISS (Facebook AI Similarity Search)
↓
5
LLM generates grounded answer
Retrieved chunks + conversation history fed to GPT. System prompt forces citations and forbids outside knowledge
GPT-3.5-turbo · ConversationalRetrievalChain
↓
6
Answer + citations returned
Response includes document name, page number, and snippet for every factual claim. Memory window keeps last 6 exchanges
ConversationBufferWindowMemory
